// 03 — Architecture

Built in 6 phases.
No backprop anywhere.

01
Foundation
STDP Learning
Spike Timing Dependent Plasticity. Neurons that fire together wire together. Weights update from spike timing alone — no gradients, no loss function.
02
Prediction
Anticipatory Firing
The network predicts the next input before it arrives. Prediction error drives learning — the brain learns by being surprised, not by being told the answer.
03
Curiosity
Emergent Curiosity
Surprise-gated learning rate. High surprise = high learning rate. The network naturally seeks unfamiliar patterns and slows down on familiar ones.
04
Adaptation
Environment Change Detection
Detects when the data distribution shifts and triggers rapid relearning. Handles non-stationary environments that destroy conventional models.
05
Hierarchy
Predictive Coding
Three-layer hierarchy. Each layer predicts the one below it. Only prediction errors propagate upward — sparse, efficient, biologically plausible.
06
Biology
E/I Balance + Homeostasis
Excitatory/inhibitory balance, homeostatic plasticity, and structural plasticity. The network maintains stable dynamics while adapting — like a real brain.

Every major AI system — GPT, Gemini, Claude — uses backpropagation. It has three unsolved problems: catastrophic forgetting, enormous energy cost, and biological implausibility. Signals do not travel backwards in real brains.

Intel built Loihi specifically because it physically cannot run backprop. They have the hardware. The software ecosystem is almost empty. I am building the software.

The architecture is already ported to Intel's Lava framework and runs on their official neuromorphic stack. The next step is testing on actual Loihi hardware.
